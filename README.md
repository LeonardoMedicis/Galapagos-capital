## 🎯 **Visão Geral**

Este repositório contém o **sistema completo de automação de conciliações** da Galapagos DTVM, desenvolvido para substituir processos manuais por soluções modernas e eficientes.

### **🚀 O que o sistema faz:**
- ✅ **Automatiza conciliações** contábeis diárias
- ✅ **Dashboard em tempo real** para monitoramento
- ✅ **Execução via interface web** ou linha de comando
- ✅ **Histórico completo** de execuções
- ✅ **Relatórios automáticos** e alertas
- ✅ **Arquitetura modular** e escalável

## 🚀 **Início Rápido**

### **⚡ Para usuários (5 minutos):**

1. **📥 Baixar repositório:**
   ```bash
   git clone https://github.com/LeonardoMedicis/Galapagos-capital.git
   cd Galapagos-capital
   ```

2. **🔧 Configuração automática:**
   - Abra `04-utilitarios/Configurador_Rapido.ipynb` no Jupyter
   - Execute todas as células
   - Pronto! Sistema configurado automaticamente

3. **📊 Usar dashboard:**
   - Abra `02-dashboard-principal/Dashboard_Principal.ipynb`
   - Execute as células
   - Dashboard abre automaticamente no navegador

### **🔧 Para desenvolvedores:**

1. **📥 Clone e configure:**
   ```bash
   git clone https://github.com/LeonardoMedicis/Galapagos-capital.git
   cd Galapagos-capital
   python 01-configuracao-ambiente/configurar_ambiente.py
   ```

2. **🚀 Execute dashboard:**
   ```bash
   python 02-dashboard-principal/executar_dashboard_adaptavel.py
   ```

3. **🌐 Acesse:**
   - http://localhost:5000
